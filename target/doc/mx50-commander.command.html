mx50-commander.command

Command wrappers which return strings per the WJ-MX50 RS232 spec.

back-color

added in MX30

(back-color red green blue)
red 0 - 255 value of red channel
green 0 - 255 value of green channel
blue 0 - 255 value of blue channel

Set background matte color using RGB instead of the native YPbPr. This command is ignored if back-color-preset is set to :bars.
