mx50-commander.control

Control MX devices, start and stop sending them commands, and access their last sent values.

clear-current

(clear-current)(clear-current device-id)(clear-current device-id cmd-id)
device-id ex. :my-mixer
cmd-id ex. :back-color

Clear the cached values for all devices, a single device, or a single command for a single device.

device

(device id)(device id params)
id ex. :my-mixer
params ex. {:port “/dev/ttyUSB0” :rate 100}

Register a device, open a port, and start listening for commands. Return a function to queue commands for execution.
